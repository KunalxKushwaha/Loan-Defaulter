# Models

This folder stores trained model files.

- `model.pkl` — the trained XGBoost + sklearn Pipeline (generated by `backend/train.py`)

To generate it, run from the project root:
```bash
python backend/train.py
```

If you already trained a model in your notebook using `joblib.dump(best_model, 'model.pkl')`,
simply copy that file here:
```bash
cp /path/to/your/model.pkl models/model.pkl
```
